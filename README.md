# E-Commerce Customer Segmentation (RFM Analysis)

### Project Overview
In this project, I analyzed 500k+ transactions from a UK-based online retail store to segment customers into actionable groups using **Python** and **SQL**.

### Tech Stack
* **Python:** Pandas, Seaborn, Matplotlib (Data Cleaning & Visualization)
* **SQL:** SQLite (Advanced CTEs and Aggregations)
* **Statistics:** RFM (Recency, Frequency, Monetary) Scoring

### Key Insights
* Identified that **Champions** make up 15% of the user base but drive 60% of total revenue.
* Discovered a large **At Risk** segment that hasn't purchased in 100+ days.

### How to Run
1. Clone the repo.
2. Run `pip install -r requirements.txt`.
3. Open `analysis.ipynb` in VS Code.
